# Vocabulary
## SDK Versions
**Minimum SDK** : <br>
States that the application will only run on mobile phones with API level *x* or higher. <br>
**Target SDK** : <br>
Refers to the version of Android you want to build for. It is recommended to keep it to the latest. <br>
**Compile SDK** : <br>
Referes to the version of Android you are currently testing on.
